# AgendaNails

Aplicação mobile em React Native para gerenciamento de agendamentos de salão de unhas.

## 📋 Funcionalidades

- Dashboard com visão geral dos agendamentos
- Gerenciamento de clientes
- Tela de configurações
- Navegação por abas

## 🛠 Tecnologias

- React Native
- TypeScript
- React Navigation

## 📦 Instalação

```bash
npm install
```

## ▶️ Como executar

```bash
npm start
```

## 📂 Estrutura do Projeto

```
src/
├── Assets/          # Imagens e recursos
├── components/      # Componentes reutilizáveis
├── routes/          # Configuração de rotas e navegação
└── screens/         # Telas da aplicação
    ├── Dashboard/
    ├── Home/
    ├── newClient/
    └── Settings/
```
